# Voice Core
**Marketing Agent Kit · Brand Brain — File 3 of 3**

Defines **how you sound.** The Marketing Agent reads this before writing every single output — it's the filter that separates your content from everyone else's.

> 📌 **How to fill this out** — This is the hardest file, because most people have never had to describe how they sound. Best approach: record yourself talking for 5 minutes about something you care about, transcribe it, and read it back. *That* is your voice — capture it here.
